Description JQuery plugin that can create a background slideshow. This plugin works differently than a carousel plugin in that the content of the element do not need to change while the background images changes.
Demo

Demo 1 :: Random background selection given a list of images for a single div element.

Demo 2 :: Multiple div elements with different transition speed and transition delay.

Demo 3 :: Background slideshow for the body sized element.

Usage 
<script type="text/javascript">
  $(function() {
    $(".bg").bgSlideShow();
  });
</script>

<body>
  <div class=".bg">Hello</div>
</body>
